# BERG | HUGO theme

## Features

- Responsive
- Dark mode
- Pagination
- Google Tag Manager
- Breadcrumbs
- Social sharing
- Meta verification

## Quick Start

```shell
hugo new site quickstart
cd quickstart
git init
git submodule add https://github.com/matusstafura/berg-hugo-theme themes/berg
echo "theme = 'berg'" >> config.toml
hugo server
```
